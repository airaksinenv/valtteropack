## Setup

1. Install [Java](https://adoptium.net/).

2. Install [Minecraft Forge 1.12.2](https://files.minecraftforge.net/net/minecraftforge/forge/index_1.12.2.html)

3. Run Minecraft once with Forge (creates folders)

3. Place [mods](mods) folder in `.minecraf/`
